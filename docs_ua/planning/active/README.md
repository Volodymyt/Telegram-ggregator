# Активні плани

Цей каталог зберігає документи активних ініціатив.

## Угода про найменування

Використовуй лаконічні назви файлів із префіксом дати та типом елемента планування одразу після дати:

- `YYYY-MM-DD-epic-short-name.md`
- `YYYY-MM-DD-story-short-name.md`
- `YYYY-MM-DD-task-short-name.md`

## Правила найменування

- Використовуй `epic`, `story` або `task` як сегмент типу.
- Решта slug має бути короткою, описовою та стабільною.
- Для посилань із квадратних дужок у backlog використовуй `file stem` без `.md`.
- Під час переміщення документа в [`archive/`](../archive/README.md) зберігай назву файлу стабільною.

## Рекомендований шаблон

```md
# Назва

Status: Active
Owner:
Last updated:

## Goal

## Scope

## Steps

## Risks

## Acceptance Criteria
```
