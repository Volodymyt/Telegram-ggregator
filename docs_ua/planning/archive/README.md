# Архів планування

Зберігай у цьому каталозі завершені, скасовані або замінені документи планування.

## Правило архівування

- Зберігай оригінальний вміст плану.
- Додавай коротку примітку на початку, якщо план було скасовано або замінено.
- Зберігай стабільні назви файлів під час переміщення документів з [`active/`](../active/README.md).
