# Active Plans

This directory stores active initiative documents.

## Naming Convention

Use concise, date-prefixed filenames with the planning item type immediately after the date:

- `YYYY-MM-DD-epic-short-name.md`
- `YYYY-MM-DD-story-short-name.md`
- `YYYY-MM-DD-task-short-name.md`

## Naming Rules

- Use one of `epic`, `story`, or `task` as the type segment.
- Keep the remaining slug short, descriptive, and stable.
- Use the file stem without `.md` when referencing the item from backlog lineage brackets.
- Keep the filename stable when moving the document into [`archive/`](../archive/README.md).

## Suggested Template

```md
# Title

Status: Active
Owner:
Last updated:

## Goal

## Scope

## Steps

## Risks

## Acceptance Criteria
```
